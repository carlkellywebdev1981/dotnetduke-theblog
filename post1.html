Starting Out In Web Development

I have always been interested and intrigued by all things tech and the world wide web in general. So when I was tasked with organising a web site for the building company I project managed for and was excited about the whole process. I contacted a few development companies and soon realised the budget I was given would not go very far. After searching for a while I came to the conclusion I would attempt to build a basic site myself using my 1 and 1 website builder. I was instantly hooked on building sites after all I've been building things my whole life. I found myself struggling with the website builder as It was hard to manipulate the elements exactly how I wanted them, my attention to detail was picking up faults in the system that looked ok to the average web surfer, but to me were not perfect and as slick as I imagined the site in my head. I built a basic site which my boss was over the moon with, but I knew I could do so much more and my thirst for knowledge into how web development worked started.
All the information and resources available for free online overwhelmed me and I began every spare minute learning or reading something, anything about programming. I started with "head first" ebooks which are well written, funny and grab your attention as to code along to the various small projects. I then moved to youtube videos following guys like Will from learning to code and Travis Neilson at devtips. Every time I search I find a new and exciting course here are some of my favourites...

  • Codecademy
  • Free code camp
  • Udemy
  • Code school
